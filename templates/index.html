🧬 Algorithm Steps (Algorithm 1)

  1. Build R: ratings + behavior → user-item matrix
  2. Build P: product features → tag-genome matrix
  3. Initialize Np random Q chromosomes ∈ [0,1]
  4. Compute RMSE fitness for each chromosome
  5. Select parents via Roulette Wheel (1/RMSE)
  6. Row-crossover: swap one user-row (prob Pc)
  7. Mutation: random gene ∈ [0,1] (prob Pm)
  8. Keep best Q (elitism), repeat Ni times
  9. Predict R̂ = Q·Pᵀ → top-N unrated items
